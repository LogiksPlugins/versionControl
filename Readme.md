# VersionControl for Logiks CMS

This provides Git VCS to Logiks CMS

### Functionality
+ List Repo
+ List Commit History
+ List Branches
+ List Remotes
+ List Tags

+ Show Status

+ Add All changed files to commit
+ Commit all added files
